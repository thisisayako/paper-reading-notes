# 論文リーディング

## 研究者情報

- 社会人大学院生（修士課程）
- 修士論文のテーマ：KAKENデータを用いたアカデミアにおける男女の生産性格差とその要因の分析

## 目的

修士論文の執筆に必要な論文を読解し、研究上の知見・論点・示唆を蓄積すること。

要約は[`論文要約`](./論文要約/)にMarkdown形式で保存する。新規作成時は[`テンプレート.md`](./論文要約/テンプレート.md)を複製して使用する。

## 論文要約の基本方針

論文要約を依頼された場合、以下のルールを必ず守ること。

- 丁寧語を使わず、文末は簡潔な体言止めとする。体言止めが難しい場合は「である調」とする
- Wordなどの文書ファイルへコピーしやすい、簡潔で統一された体裁とする
- 正確性と実用性を最優先する
- ユーザーを気持ちよくするためのお世辞を言わない
- 最高の回答に必要な情報が不足している場合、回答の生成前に、些細な点も含めて必ず質問する
- 内容への確信度が90%以上の場合のみ回答し、それ未満の場合は「分からない」と明記する
- 英語の文書を渡された場合は、英語で内容を検討したうえで回答する
- 倫理に関する助言は必要最小限にとどめる
- 論文に記載された内容と、要約者による推論・批判的検討を明確に区別する
- 数値、サンプル数、分析対象、期間、変数、推定方法、図表番号などを可能な限り原文と照合する

## 論文要約の出力形式

以下の見出しと順序で要約すること。

# 背景・目的

研究の背景、先行研究上の位置づけ、研究課題、仮説、分析目的の整理。

# 方法

使用データ、分析対象、サンプル、対象期間、主要変数、指標の定義、研究デザイン、推定方法の整理。

# 結果・考察

論文中の重要な図表に沿った主要結果の整理。各図表について、分析内容、主要な数値・傾向、著者の解釈、修士論文への示唆を記載。

# 主張の限界・残された課題

論文に記載された主要な限界に加え、論文に明記されていない課題も検討。通常の研究デザインであれば取り組むと考えられる事項、因果推論上の問題、測定上の問題、外的妥当性、頑健性検証、代替説明などの観点から整理。論文に記載された限界と、要約者が追加で指摘する課題を区別。

# 貢献・採録理由

既存研究との差分と学術的貢献の整理。新規性のある問い、データ、指標、方法、結果、理論的・実務的含意などを特定し、どのような差分・貢献がジャーナル等への採録につながったと考えられるかを検討。採録理由は、明示的な事実ではなく要約者の推論である場合、その旨を明記。
