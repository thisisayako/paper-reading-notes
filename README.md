# 論文リーディング

## 研究者情報

- 社会人大学院生（修士課程）
- 修士論文のテーマ：KAKENデータを用いたアカデミアにおける男女の生産性格差とその要因の分析

## 目的

修士論文の執筆に必要な論文を読解し、研究上の知見・論点・示唆を蓄積すること。

要約は[`論文要約`](./論文要約/)にMarkdown形式で保存する。新規作成時は[`テンプレート.md`](./論文要約/テンプレート.md)を複製して使用する。

## 作業手順

論文の要約作業は、次の順序で一連の作業として実施する。

1. 論文を読み、[`論文要約`](./論文要約/)にMarkdown要約を作成する
2. 要約ファイルの `##書誌情報` から出版年、DOI、URL を拾う
3. 要約ファイルの `##背景・目的` から背景・目的を抽出する
4. 要約ファイルの `##結果・考察` から、主要図表ごとの結果・考察を抽出する
5. Google スプレッドシートの「論文要約一覧」を更新し、少なくとも次の項目を埋める。
   - No.
   - 著者
   - 出版年
   - タイトル
   - DOI・URL
   - 目的
   - 結果
   - 研究への示唆
   - 既存の要約データが残っているため、必ず空欄行から追記し、既存行は上書きしない
6. 「研究への示唆」には、科研費データを用いたアカデミアにおける男女の生産性格差の分析にどう活かせるかを、後から読み返して分かる程度に具体的に記す

Markdown要約の作成と一覧表の更新は、別作業ではなく同じ作業単位として扱う。
この作業手順に該当する場合は、都度の許可確認を挟まず自動で進める。

## 論文要約の基本方針

論文要約を依頼された場合、以下のルールを必ず守ること。

- 丁寧語を使わず、文末は簡潔な体言止めとする。体言止めが難しい場合は「である調」とする
- Wordなどの文書ファイルへコピーしやすい、簡潔で統一された体裁とする
- 正確性と実用性を最優先する
- ユーザーを気持ちよくするためのお世辞を言わない
- 最高の回答に必要な情報が不足している場合、回答の生成前に、些細な点も含めて必ず質問する
- 内容への確信度が90%以上の場合のみ回答し、それ未満の場合は「分からない」と明記する
- 英語の文書を渡された場合は、英語で内容を検討したうえで回答する
- 倫理に関する助言は必要最小限にとどめる
- 論文に記載された内容と、要約者による推論・批判的検討を明確に区別する
- 数値、サンプル数、分析対象、期間、変数、推定方法、図表番号などを可能な限り原文と照合する

## 論文要約の粒度

[`2018_Hechtman_NIH-funding-longevity-by-gender.md`](./論文要約/2018_Hechtman_NIH-funding-longevity-by-gender.md)を、要約の詳しさ・分析粒度・記述形式の基準とする。

- 書誌情報、BibTeX、背景・目的、方法、結果・考察、限界・課題、貢献・採録理由、修士論文への示唆を記載する
- 背景・目的では、先行研究上の位置づけ、研究上のギャップ、中心的な問い、アウトカムの意味を整理する
- 方法では、データ、分析対象、対象期間、サンプル数、変数の定義、推定方法、比較群、感度分析を、研究デザインを再現できる程度に記載する
- 結果・考察では、主要な図表ごとに小見出しを設け、図表単位で結果と考察を分けて記載する。図表が複数ない論文でも、主要な分析単位ごとに小見出しを切る
  - 図表が検証する内容
  - 主要な数値、効果量、信頼区間、P値等
  - 統計的有意性と実質的な差の大きさ
  - 著者による解釈
  - 要約者による解釈と注意点
  - 修士論文への示唆
- 統計手法や指標は名称を列挙するだけでなく、その結果が何を意味するかを簡潔に説明する
- 「有意差なし」を「差がないことの証明」と扱わず、検出力、追跡期間、標本選択などを考慮する
- 著者が記載した限界と、要約者による追加の検討を分ける
- 論文に記載された貢献と、要約者が推論した採録理由を分ける
- 修士論文への示唆では、KAKENデータへの応用可能性を、変数、分析単位、推定方法、仮説、注意すべきバイアスの水準まで具体化する
- 論文の情報量や重要性に応じて分量は調整するが、重要な分析結果や留保を短文化のために省略しない

## 論文要約の出力形式

以下の見出しと順序で要約すること。

# 書誌情報

著者、出版年、掲載誌、巻号、ページ、DOI・URL、読了日、キーワード、BibTeXを記載。

# 背景・目的

研究の背景、先行研究上の位置づけ、研究課題、仮説、分析目的の整理。

# 方法

使用データ、分析対象、サンプル、対象期間、主要変数、指標の定義、研究デザイン、推定方法の整理。

# 結果・考察

論文中の重要な図表または主要分析単位に沿った主要結果の整理。各単位ごとに小見出しを付け、その下で「結果」と「考察」を分けて記載する。結果では、分析内容、分析に用いた手法、主要な数値・傾向、統計的有意性を簡潔にまとめ、考察では、著者の解釈、要約者による解釈と注意点、修士論文への示唆を記載する。Hechtman 論文のように、図表番号や分析単位が追える粒度を優先する。

# 主張の限界・残された課題

論文に記載された主要な限界に加え、論文に明記されていない課題も検討。通常の研究デザインであれば取り組むと考えられる事項、因果推論上の問題、測定上の問題、外的妥当性、頑健性検証、代替説明などの観点から整理。論文に記載された限界と、要約者が追加で指摘する課題を区別。

# 貢献・採録理由

既存研究との差分と学術的貢献の整理。新規性のある問い、データ、指標、方法、結果、理論的・実務的含意などを特定し、どのような差分・貢献がジャーナル等への採録につながったと考えられるかを検討。採録理由は、明示的な事実ではなく要約者の推論である場合、その旨を明記。
