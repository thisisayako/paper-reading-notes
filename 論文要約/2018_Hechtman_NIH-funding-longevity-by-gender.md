## 背景・目的

生物医学分野では博士号取得者に占める女性割合が男性と同程度に達している一方、ポスドク、助教、教授へと進むにつれて女性割合が低下する「leaky pipeline」が指摘されてきた。NIHの新規研究助成申請と研究助成受給者に占める女性は3分の1未満であるが、初回助成の獲得成功率には大きな男女差がないという先行知見も存在。

本論文の目的は、最初の主要なNIH Research Project Grant（RPG）を獲得した研究代表者に対象を限定し、その後の助成継続期間に男女差があるかを検証すること。加えて、助成件数・金額・安定性、新規申請と更新申請の頻度、採択率、審査スコア、研究者属性を比較し、助成継続期間の男女差と関連する要因を検討。

中心的な問いは、独立研究者として最初の大型研究費を得た女性が、その後も男性と同程度にNIH助成を維持するかという点。助成継続をアカデミアでの研究参加・キャリア成功の代理指標として使用。

## 方法

- 使用データ：NIHの内部行政データベースIMPACIIに収録された申請・採択記録。公開可能な助成記録はNIH RePORTERから取得可能
- 分析対象：1991-2010年に初めて主要RPGを獲得した研究代表者34,770人。男性24,110人、女性10,660人。性別報告率98.5％
- 対象期間：助成履歴の分析期間は1991-2015年。再参入判定のため、2017年までの記録を利用
- 対象助成：R01、U01、P01など、NIHが主要研究プロジェクトと位置づけるRPG
- 主要アウトカム：最初の主要RPG獲得年からNIH助成プールを退出するまでの期間。最後の主要RPG終了後に3年以上助成がない場合を退出と定義。2015年にも助成を受けていた研究者は右打ち切り。右打ち切りは約45％
- 助成特性：年間平均保有プロジェクト数、Biomedical Research and Development Price Indexで実質化した年間平均助成額、助成期間、助成年数、対象期間中に助成を保有した年の割合
- 申請行動・審査結果：年間新規申請数、新規申請採択率、新規申請の平均スコア・パーセンタイル、更新可能案件に対する更新申請率、更新申請採択率、更新申請の平均スコア・パーセンタイル
- 研究者属性：初回採択時年齢、学位種別、最初のRPG種別、最初のNIH助成年、所属機関のCarnegie分類、所属機関のNIH助成水準
- 推定方法：Kaplan-Meier法、Greenwood法による95％信頼区間、Mantel-Haenszel検定、Gehan-Wilcoxon検定、Wilcoxon順位和検定。補足分析としてRandom Survival Forest（1,000本、Harrellの一致指数0.82超）と傾向スコア最近傍マッチング
- コホート区分：初回主要RPG獲得年により1991-1995年、1996-2000年、2001-2005年、2006-2010年の4群

## 結果・考察

### 図1：NIH助成の継続期間

図1Aでは、男性の助成継続率が女性をわずかに上回る結果。全期間の生存曲線の差は統計的に有意であり、Mantel-Haenszel検定はχ²(1) = 15.5、P < 0.001、Gehan-Wilcoxon検定はχ²(1) = 10.2、P = 0.001。25年時点の男女差は3.5パーセントポイントであり、差の規模は小さい。

女性は分析対象者の30.66％にとどまり、初回主要RPG獲得者となる段階で大幅に過少代表。著者は、初回助成獲得後に女性が一貫して急速に退出するという単純なleaky pipeline像とは整合しない結果と解釈。

この結果を理解するには、「初回助成を得るまで」と「初回助成を得た後」を分ける必要がある。本研究の対象者が博士号を取得したと推定される1979-1999年には、関連する生物医学分野の博士号取得者に占める女性は33-53％、全体では約45％であった。一方、初回主要RPG獲得者に占める女性は30.66％にとどまる。したがって、博士号取得から初回大型助成の獲得に至るまでの段階で、女性割合が大きく低下していたことを意味する。

これに対して、初回RPGを獲得した後の25年間における助成継続率の男女差は3.5パーセントポイント。統計的には有意であるが、女性だけが男性より急速に助成プールから退出し続けていると解釈できるほど大きな差ではない。すなわち、女性割合の大きな低下は初回大型助成を得る以前またはその獲得段階で生じ、その後の助成継続における男女差は比較的小さいという構造。

この結果はジェンダー格差の不存在を意味するものではない。また、分析対象はすでに初回RPGを獲得した研究者に限定されており、助成獲得までの強い選抜を通過した女性と男性の条件付き比較である。そのため、この結果を女性研究者全体のキャリア継続へ一般化することには注意が必要。

図1Bでは、初回助成から同じ年数が経過した時点で比較すると、後年に初回助成を得たコホートほど早く助成プールから退出する傾向。例えば、初回助成から10年後の継続率を比べると、1990年代前半のコホートより2000年代のコホートのほうが低い傾向であり、後年の研究者ほど初回助成後の比較的早い段階で3年以上の無助成状態に入りやすいことを意味する。

4コホート間の差はMantel-Haenszel検定でχ²(3) = 223、P < 0.001、Gehan-Wilcoxon検定でχ²(3) = 217、P < 0.001。自由度が3であるのは4コホートを比較しているため。P < 0.001は、4群の助成継続曲線が実際には同じであると仮定した場合、観察された程度以上の差が偶然に生じる確率が0.1％未満であることを示す。Mantel-Haenszel検定は追跡期間全体の差を比較し、Gehan-Wilcoxon検定は追跡初期の差を相対的に強く評価する。両検定で有意であるため、特定の一時点に限らず、継続曲線全体にコホート差が存在するとの判断。

著者は、後年ほどNIH助成の獲得競争が激しくなり、初回助成後の更新や次の助成の獲得が難しくなった可能性を指摘。ただし、この分析は競争激化の因果効果を直接推定したものではなく、研究者属性、助成制度、対象助成の構成、申請行動などのコホート差も含む結果。Kaplan-Meier法は追跡期間の違いと右打ち切りを考慮するため、後年コホートの観察期間が短いことだけで曲線が低くなるわけではない。一方、後年コホートでは長期の継続率を観察できないため、比較可能な追跡期間が短いという制約。

### 図2：コホート別の男女差

1991-1995年コホートでは女性2,192人、男性5,539人であり、女性の退出率がわずかに高い結果（MH：P = 0.01、GW：P = 0.05）。1996-2000年コホートでも女性2,436人、男性5,876人で同様の小差（MH：P = 0.03、GW：P = 0.05）。

2001-2005年コホートでは女性2,690人、男性6,068人で有意差なし（MH：P = 0.14、GW：P = 0.31）。2006-2010年コホートでも女性3,342人、男性6,627人で有意差なし（MH：P = 0.68、GW：P = 0.74）。女性割合は28.35％、29.31％、30.71％、33.52％と後年ほど上昇。

退出を「5年以上の無助成」に変更するとコホート別の男女差は有意でなくなる一方、全標本での小さな男女差は残存。コホート別結果は退出定義に敏感であり、頑健な結論は全体差が小さいという点。

### 助成件数・金額・安定性

女性の年間平均保有プロジェクト数は1.17件、男性は1.24件（P < 0.001）。女性の年間平均助成額は316,628ドル、男性は343,496ドル（P < 0.001）であるが、1プロジェクト当たりの平均助成額には有意差なし（女性269,527ドル、男性282,337ドル、P = 0.60）。総額差の中心は単価ではなく保有件数の差。

女性の助成期間は平均10.08年、男性は10.56年、実際の助成年数は女性9.28年、男性9.77年であり、ともに小さな有意差。助成期間中に実際に助成を保有した年の割合は女性94.45％、男性94.62％で有意差なし（P = 0.72）。女性により長い無助成ギャップがあるという証拠は不在。

### 図3：新規プロジェクト申請

女性の年間新規申請数は0.38件、男性は0.45件であり、全コホートで女性が少ない結果（全体P < 0.001）。一方、新規申請の平均採択率は男女とも24.67％で有意差なし（P = 0.19）。例外は1991-1995年コホートで、女性28.68％、男性30.65％の小差（P = 0.04）。

平均審査スコア・パーセンタイルは女性25.77、男性26.00で全体差なし（P = 0.37、低いほど高評価）。2006-2010年コホートでは女性25.56、男性26.48で女性がわずかに高評価。新規申請では審査上の一貫した不利よりも、女性の申請件数の少なさが主要な差。

### 図4：競争的更新申請

更新可能案件に対する申請率は女性42.45％、男性45.44％であり、全コホートで女性が低い結果（全体P < 0.001）。更新申請の採択率も女性35.98％、男性39.28％で女性が低く（P < 0.001）、最も新しいコホートを除いてコホート別にも有意差。

更新申請の平均スコア・パーセンタイルは女性23.76、男性23.00で、全体では女性が低評価（P = 0.01）。ただし各コホート内では有意差なし。主要RPGは通常4-5年で終了するため、女性の更新申請率と更新採択率の低さは、初回助成から5-6年付近で生じる小さな継続率差と整合。

### 研究者属性と調整分析

女性はPhDが73.7％、MDが14.3％、MD/PhDが9.1％、男性はそれぞれ63.6％、19.2％、15.4％。助成継続期間が比較的長いMD/PhDに女性が少ないこと、女性が競争の激化した後年コホートに多いことが、未調整の男女差に関係する可能性。

初回助成時の研究者属性で男女をマッチングすると、女性10,212人と男性10,212人の助成継続率に差なし（MH：P = 0.63、GW：P = 0.82）。Random Survival Forestでは、助成継続の予測に最も重要な変数が更新申請率（VIMP = 0.10）、年間新規申請数（VIMP = 0.08）、年間助成額（VIMP = 0.04）であり、性別の予測力は最小（VIMP < 0.001）。更新申請率と初回助成年でマッチングした分析でも男女差なし（各9,395人、MH：P = 0.60、GW：P = 0.20）。

著者の総合的解釈は、初回主要RPG獲得後の助成継続期間の男女差は小さく、その差は初期属性と申請行動の双方に関連するというもの。最大の格差は初回RPG獲得後の退出ではなく、女性が初回RPG保有者となる以前またはその段階で生じているとの結論。

## 主張の限界・残された課題

### 著者が記載した限界

- NIH行政データだけでは、女性のキャリア移行や申請行動を左右する社会的要因を把握不能
- 女性が新規申請・更新申請を男性と同程度に提出しない理由について、キャリア変更、申請戦略、その他の要因を識別不能
- 更新申請で女性の審査結果が低い理由を本研究では解明しておらず、追加研究が必要
- 2015年時点で助成を保有する研究者は退出時点を確定できず、約45％が右打ち切り

### 要約者による追加の検討

- 初回主要RPG獲得者に条件づけた分析であり、助成獲得前の大きなジェンダー格差を直接説明する設計ではない。初回採択を選抜条件とすることで、採択者内の男女比較に選択バイアスが生じる可能性
- 「NIH助成プールからの退出」は、アカデミアや研究職からの退出と同義ではない。他機関の助成への移行、教育・臨床への重点変更、研究費を必要としない活動などを離職として誤分類する可能性
- 性別を男女二群で扱い、性自認の多様性や人種・民族、親役割、分野、機関属性との交差性を十分に分析していない点
- 更新申請率は助成継続の過程で決まる媒介変数であり、これにマッチングして男女差が消えることは因果効果の証明ではない。「女性の申請を増やせば格差が解消する」と直ちに解釈することは困難
- 申請数の差自体が、研究資源、共同研究ネットワーク、教育・サービス負担、家族責任、過去の審査経験など未観測要因の結果である可能性
- 大標本のため小さな差でも統計的に有意になりやすい。P値だけでなく、25年時点3.5ポイントなど効果量の実質的重要性を重視すべき設計
- 後年コホートほど観察期間が短く、助成環境も変化しているため、コホート間比較には期間効果と追跡期間の差が混在
- 退出を3年または5年の無助成で定義した結果が一部異なるため、結論の一部が操作的定義に依存
- NIH内部データに依存し、申請単位の完全な再現データが一般公開されていない可能性。米国の生物医学研究とNIH主要RPGに限定され、他国、他分野、他の助成機関への外的妥当性は限定的

## 貢献・採録理由

### 論文に記載された貢献

- 34,770人を最長25年間追跡する大規模行政データを用い、初回主要研究費獲得後の助成継続期間を男女別に初めて検討した点
- 女性がキャリアの全段階で男性より急速に退出するという単純なleaky pipeline像を修正し、初回大型助成獲得後の継続率差が小さいことを提示した点
- 助成継続だけでなく、新規・更新申請、採択率、審査スコア、助成件数・金額、研究者属性を結びつけ、格差が現れる局面を特定した点
- 女性の過少代表が最も大きいのは初回RPG獲得者となる以前またはその段階であり、獲得後には更新申請が重要な移行点であることを示した点

### 要約者による採録理由の推論

以下は論文に明記された査読判断ではなく、要約者による推論。

PNASでの採録につながったと考えられる要因は、政策的重要性の高いジェンダー格差を、NIH内部の大規模・長期行政データで検証した点。通説的なleaky pipelineの一様な適用に反する結果を示しつつ、格差の存在自体を否定せず、初回助成前と更新時という介入可能な局面を具体化した点も重要。生存分析、コホート比較、マッチング、機械学習的生存モデルを組み合わせた多面的検証と、NIHの資金配分政策へ直結する含意も採録上の強みと推察。

## 修士論文への示唆

- 男女の生産性格差をキャリア全体の一様な差として扱わず、参入、初回獲得、継続、更新、退出という段階別に分解する必要性
- KAKENデータでも、初回採択者に限定した継続期間分析と、研究者全体を母集団とする初回採択分析を区別する設計が有効
- 採択率だけでなく、申請頻度、更新申請率、保有課題数、配分額、無採択期間を別々のアウトカムとして分析する必要性
- 「女性は採択されにくい」だけでなく、「申請数・更新率が異なる理由」を研究資源、職位、分野、所属機関、共同研究、ライフイベントなどから検討する必要性
- 初回採択者への条件づけによる選択バイアスと、申請行動を統制・マッチングする際の媒介変数調整に注意が必要
- Kaplan-Meier曲線や離散時間ハザードモデルを用いた研究費継続分析、採択間隔の定義を変えた感度分析、初回採択年コホート別分析がKAKEN研究へ応用可能
