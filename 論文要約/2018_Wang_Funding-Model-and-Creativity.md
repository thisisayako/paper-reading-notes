## 背景・目的

研究費制度は block funding から competitive funding へ移る傾向にあるが、それが創造性を高めるのかは不明である。本論文は、日本のデータを用いて、助成形態と研究成果の新規性の関係を検証する。

特に、研究者の status によって競争的助成の効果が変わるかを問う。

## 方法

- 使用データ：日本の研究プロジェクトに関する調査データと bibliometric 情報
- 分析対象・サンプル：研究プロジェクトとそこから生まれた論文
- 対象期間：日本の研究費制度が競争化する時期を含む期間
- 主要変数・指標：novelty、funding type、status、gender
- 研究デザイン・推定方法：調査データと文献計量の結合、比較分析

## 結果・考察

### 図1：資金モデルと新規性

#### 結果

競争的資金で支援されたプロジェクトのほうが、平均的には block funding より新規性が高い。ただし、そのプラス効果は高 status の研究者に限られ、低 status の研究者、とくに junior や女性では逆に新規性と負に関連する。

#### 考察

競争的資金は平均では創造性を押し上げるが、受益者の属性によって効果が反転する。制度は資源を与えるだけでなく、誰に自由と余裕を与えるかを決める。

### 図2：高 status と低 status の異質性

#### 結果

高 status 研究者は自由度を得て新規性を高めるが、低 status 研究者は審査対応や短期成果への圧力により保守化する可能性がある。

#### 考察

同じ funding model でも、研究者の属性によってアウトカムがまったく異なる。平均効果だけで制度を評価すると、弱い立場の研究者への逆効果を見落とす。

## 主張の限界・残された課題

#### 考察

この論文の含意は、競争的資金が自動的にイノベーションを生むわけではないということにある。制度は、資源を与えるだけでなく、誰に自由と余裕を与えるかを決めてしまう。

したがって、創造性を高めたいなら、競争的配分だけでなく、低 status 研究者が挑戦的テーマに取り組める安全地帯も必要になる。資金制度は、平均効果だけでなく分配効果を見て設計すべきだ。

### 著者が記載した限界

- 観察研究であり、funding model の因果効果を完全には識別できない

### 要約者による追加の検討

- novelty 指標は bibliometric に依存し、創造性の一部しか捉えない
- status と gender が相互作用するため、単純平均では見えにくい

## 貢献・採録理由

### 論文に記載された貢献

- block funding と competitive funding を同時比較した点
- status contingency と gender の相互作用を示した点

### 要約者による採録理由の推論

資金配分制度の設計が創造性にどう効くかを、日本データで具体化した点が重要。政策論への直結性が高い。

## 修士論文への示唆

KAKENでも、採択率だけでなく、助成形態や研究者の status による効果の異質性を見たい。女性が低 status に偏るなら、競争的資金の効果が不利に働く可能性がある。
